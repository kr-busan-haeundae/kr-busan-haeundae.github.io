# 해운대구 · 부산광역시

> 해운대구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-haeundae.github.io/**

해운대구는 부산 최대의 관광 브랜드를 품은 자치구로, 백사장 길이 약 1.5km의 해운대해수욕장과 국제 비즈니스·주거 신도심 센텀시티·마린시티가 공존하는 곳입니다. 여름 성수기에는 하루 수십만 명이 백사장을 찾아 전국에서 가장 붐비는 해수욕장이라는 명성을 이어가고 있습니다. 매년 10월 열리는 부산국제영화제(BIFF)의 중심 무대인 영화의전당이 이곳에 자리해 아시아를 대표하는 영화 도시의 얼굴 역할을 합니다. 초고층 마천루가 늘어선 마린시티와 세계 최대 규모의 백화점으로 기네스에 오른 신세계백화점 센텀시티점은 해운대의 도회적 스카이라인을 상징합니다. 동백섬 누리마루APEC하우스에서 광안대교까지 이어지는 야경은 국내외 관광객이 손꼽는 부산의 대표 풍경입니다. 바다와 마천루, 영화와 축제가 한데 어우러진 해운대는 단순한 해변 휴양지를 넘어 부산의 국제 관문으로 성장했습니다.
